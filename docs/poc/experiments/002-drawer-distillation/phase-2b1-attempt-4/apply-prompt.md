# Phase 2B1 corrective application prompt

Use the supplied source-blind manifest and application packet as application inputs only. Preserve profile precedence. This prompt does not itself create or prescribe an implementation. Under an explicit separate implementation task, the supplied inputs may be used to create a neutral static reconstruction.

If an owner-supplied distinct secondary collection and caption are present, a quiet secondary caption may be rendered for that collection only. Leave primary navigation uncaptained. Render leading icons only when their complete optional semantic inputs are supplied; otherwise omit them without a generic fallback.

Record any missing owner inputs as omissions or decisions needed; do not invent captions, icon meaning, assets, destinations, state, or behavior.
