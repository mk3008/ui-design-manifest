# Guidance

Keep the workspace calm and utilitarian: light surfaces, modest separation between regions, compact but readable controls, and clear type hierarchy. Use a quiet lead area to introduce current content before denser explanatory sections.

# Unresolved

Exact values, contrast measurements, typography families, motion, and state treatments are target-product decisions.
