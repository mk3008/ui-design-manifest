# Review checks

| Check | Pass condition |
| --- | --- |
| Independent axes | Every schema axis has an allowed classification. |
| Hidden ownership | A fully hidden Drawer has no sole Drawer-owned reopen control. |
| Rail ownership | An icon rail may retain one declared Drawer-owned expansion control and declared shortcuts. |
| Controller clarity | Exactly one authoritative controller is declared for every exercised state. |
| Policy separation | Required patterns, optional adaptive mapping, and always-visible/no-toggle are separate cases. |
| Accessibility | Each declared control has a discernible name and declared expanded/controls relationship. |
| Restraint | No breakpoint, focus, keyboard, motion, persistence, backdrop, presentation, or displacement is inferred. |
| Source boundary | The Implementer bundle contains no source identity, URL, screenshot, evidence record, local source path, or historical record link. |
