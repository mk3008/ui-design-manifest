# Prompt

Given one bounded static condition-entry region, classify every observation-schema axis. Keep source-aware evidence separate. Produce a compact Manifest, Application Input Contract, and apply instruction that preserve only visible group responsibility, labelled control association, control roles, reading order, compact rhythm, action responsibility, action-group alignment, and action order. When Clear or its order is not visible, use the authored default Search then Clear and label it as a default rather than an observation. State every other non-visible behavior as unresolved. Do not add product meaning, copied prose, unlabeled controls, results, or responsive rules.
