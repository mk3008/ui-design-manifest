# Corrected abstraction decisions

- A navigation caption is not universal. Primary navigation is uncaptained; a caption is optional and requires an owner-supplied distinct secondary collection and caption.
- A leading icon is not universal. It requires an owner-supplied semantic slot, permitted asset or category, accessible-name relationship, and display condition.
- Missing caption or icon inputs result in omission of the dependent element. No generic marker, square, control-like fallback, or icon set is permitted.

# Application input result

The fixture supplies no collection role or caption metadata, so caption rendering is not exercised. It supplies no semantic icon slots, so no leading markers are included.

# Boundary

This record is guidance and application input only. It authorizes no implementation, runtime dependency, package, or external asset.
