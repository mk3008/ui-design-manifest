# Drawer source-blind bundle

This bundle supplies a bounded, source-independent input for a static Drawer reconstruction and later semantic round-trip review.

- [Manifest](manifest.md) — retained Drawer intent and known limits.
- [Application Input Contract](application-input-contract.md) — semantic inputs an application owner must provide or explicitly omit.
- [Fixture trace](fixture-trace.md) — neutral values permitted only for the declared static exercise.
- [Claim matrix](claim-matrix.md) — material claims and the evidence needed to evaluate them.
- [Exercised classification](exercised-classification.md) — what this pre-implementation bundle does and does not exercise.
- [Apply packet](apply-packet.md) — source-blind handoff for a renderer.
- [Extraction and evaluation record](extraction-evaluation-record.md) — non-identifying observation boundary and outstanding questions.
