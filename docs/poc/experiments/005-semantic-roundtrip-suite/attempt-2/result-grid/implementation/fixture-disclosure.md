# Fixture disclosure for independent re-extraction

The static evidence uses only the authorized neutral forms:

| Contract slot | Rendered neutral form | Boundary |
| --- | --- | --- |
| Region header | `Header-I` | Structural token only. |
| Record title | `Record-I`, `Record-II` | Arbitrary fixture instances; not identities. |
| Supporting metadata | `Metadata-I`, `Metadata-II` | Arbitrary fixture instances; not product fields or values. |
| Marker | `Marker-I`, `Marker-II` | Structural association only; not state or status. |
| Optional destination | absent | No affordance or action is rendered. |

The evidence renders a header preceding a repeated entry region. Each entry groups its title, lower-priority metadata, and an associated non-semantic marker. It makes no claim about source records, quantity, ordering, semantics, behavior, or preservation.
