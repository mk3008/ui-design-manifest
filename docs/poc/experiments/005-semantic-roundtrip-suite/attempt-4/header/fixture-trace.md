# Fixture

Use a short neutral identity title. Provide one neutral navigation item, one separately grouped search entry, and one utility icon control with a meaningful inline SVG and accessible name. The fixture must visibly show title prominence, group order, compact density, and a Header surface with a lower edge relation.

# Exclusions

Do not supply page-local context, breadcrumbs, local navigation, Drawer body content, a Drawer trigger, target side, state, persistence, or behavior. These are not exercised because the Contract does not supply them.

# Trace

Fixture values are explicitly declared test fixtures for already-defined semantic slots. They introduce no product identity, destination, count, status, or new hierarchy slot.
