# Purpose

Use this method to preserve a condition group's responsibility, labelled controls, ordering, density, and action ownership without importing product vocabulary or inventing behavior. It is a calibration method, not a fidelity or validation claim.

# Procedure

1. Bound the region and exclude surrounding navigation, results, and unrelated content.
2. Complete every axis in the [observation schema](observation-schema.md) as `observed`, `not observed`, `unresolved`, or `not applicable`.
3. Record source-aware evidence only in an observation record; derive the source-blind handoff from the [extraction template](extraction-template.md).
4. Use the [rubric](first-pass-rubric.md) before supplying the Manifest, Application Input Contract, and apply instruction to an implementer.

# Boundary

Static evidence establishes visible structure only. It cannot establish interaction, conditional logic, validation, focus, keyboard behavior, motion, persistence, or responsive transformation.
