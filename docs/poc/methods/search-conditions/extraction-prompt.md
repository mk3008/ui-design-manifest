# Prompt

Given one bounded static condition-entry region, classify every observation-schema axis. Keep source-aware evidence separate. Produce a compact Manifest, Application Input Contract, and apply instruction that preserve only visible group responsibility, labelled control association, control roles, reading order, compact rhythm, and primary action ownership. State every non-visible behavior as unresolved. Do not add product meaning, copied prose, unlabeled controls, placeholder-only labels, secondary actions, results, or responsive rules.
