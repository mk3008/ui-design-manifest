# Experiment 001: Workspace shell extraction

This Gate 2 experiment documents one bounded desktop workspace shell from permitted visible UI evidence. It is an extraction artifact, not a reconstruction or a claim of visual fidelity.

## Scope

Included: the first 1280x720 desktop viewport in light theme with JP active, Overview selected, page scroll at the top, and the sidebar first expanded and then collapsed through its visible toggle. The visual scope is the application header, visible sidebar portion, Overview lead card, and Overview content visible in that viewport.

Excluded: every other page, navigation action, language or theme switch, file operation, sidebar scrolling, mobile or narrow viewports, focus variants, lower-page content, source code, styles, network activity, storage, and implementation details.
