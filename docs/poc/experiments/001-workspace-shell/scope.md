# Scope statement

## Included source states

1. Initial 1280x720 desktop viewport; light theme; JP selected; Overview selected; page scroll at the top; sidebar expanded.
2. The same viewport and state after activating the visible sidebar-toggle control to collapse the sidebar, followed by restoration of the expanded state.

The visible application header, expanded sidebar portion, Overview lead card, and Overview content within that first viewport are the only visual targets.

## Exclusions

No other controls were activated. No pages, menu destinations, language switching, theme switching, load/save flows, sidebar scrolling, narrow/mobile viewports, focus variants, or below-the-fold content were inspected. No source files, scripts, styles, network requests, storage, cookies, computed declarations, or hidden implementation material were inspected.
