# Guidance

Use a dark, self-contained navigation surface with a calm visual rhythm. Separate major context blocks with restrained dividers; reserve stronger emphasis for the current navigation location.

Place primary destinations in a vertical reading order. When a destination has children, make the parent-child relationship visible through indentation and a disclosure affordance. Keep peer rows visually consistent so that hierarchy, rather than decoration, explains their relationship.

Use group labels to introduce secondary collections only after the primary path. Keep the labels quieter than actionable rows.

# Unresolved

The required labels, destination taxonomy, and any record-specific meaning are application inputs, not supplied by this guidance.
