# Drawer revalidation bundle

This source-blind implementer bundle records bounded guidance for an open primary navigation Drawer at a wide viewport. Start with [the Manifest](manifest.md), then use the [Application Input Contract](application-input-contract.md) and [fixture trace](fixture-trace.md) together. The [claim matrix](claim-matrix.md) and [coverage classification](exercised-classification.md) define the reconstruction review obligations.

The bundle is directional guidance. Current product implementation and explicit requirements take precedence. Unsupported visibility, responsive, persistence, and interaction details remain unresolved.
