# Source-blind application packet

An implementer may receive this file only, together with the standard application prompt and target-product inputs.

## Abstracted header direction

Use a compact global header only when the target task establishes its responsibility. Make global controls, product context, and local navigation visually separable without prescribing original-specific labels, actions, branding, geometry, or responsive behavior. Preserve unresolved behavior as a gap rather than deriving it from an original UI.

Do not provide calibration, holdout, contract, capture, review, or extraction material. They are test-only.
