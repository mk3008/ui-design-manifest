# Drawer design manifest

This bundle contains source-blind guidance for a visible open navigation drawer.

- [Manifest](manifest.md) defines scope and precedence.
- [Foundations](foundations/index.md) describes grouping and reading order.
- [Components](components/index.md) describes the drawer and navigation states.
- [Policies](policies/index.md) defines application-input and fixture constraints.
- [Screen patterns](screen-patterns/index.md) records the bounded open-drawer pattern.
